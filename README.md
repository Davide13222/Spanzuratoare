# Spânzurătoarea

Joc de spânzurătoarea în română.

## Fișiere
- index.html
- style.css
- script.js

## Cum pornești jocul
1. Pune toate 3 fișierele în același folder
2. Deschide index.html în browser

## Cum se joacă
1. Alegi: ușor sau greu
2. Ghicești litere
3. Greșit = pierzi o viață
4. Câștigi dacă ghicești tot cuvântul
